Ruby Mendicant University Exam #1

I had so much fun with this project, that I decided to add a little bit of analysis for kicks. I ran my implementation several hundred times in JRuby 1.6.4 and Ruby 1.9.2 to see the memory usage in each Ruby at different character insertion thresholds. However, don't read too much into the stats, they aren't really indicative of much.

Of course, Ruby 1.9.2 used less memory overall, but the really neat thing to see was: as the threshold increased, JRuby's memory increase was nowhere near as dramatic as Ruby 1.9.2's memory increase.
